# Política de privacidad — para Play Console

Play Console pide una **URL pública** de política de privacidad. Usa:

```
https://st885.github.io/trexo-roll/privacy.html
```

- El archivo fuente es `privacy.html` en la **raíz** del repo; GitHub Pages lo sirve en esa
  URL automáticamente (tras el deploy de la web).
- La versión markdown está en `docs/privacy-policy.md` y los términos en `docs/terms.md`.

## Data Safety (resumen para el formulario de Play)
- **¿Recopila o comparte datos del usuario?** No se envía nada fuera del dispositivo.
- **Datos almacenados solo en el dispositivo**: progreso del juego y nombre de perfil local.
- **Sin** publicidad, **sin** analítica, **sin** ubicación, **sin** permisos sensibles.
- **Contraseñas**: no se recopilan (acceso = perfil local).
- Cifrado en tránsito: N/A (no hay tránsito de datos en esta versión).
- Eliminación de datos: el usuario puede borrar todo desde Ajustes → Reiniciar progreso.
